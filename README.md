# Homebrew Tap für meradOS Tools

## Installation

```bash
brew tap maf4711/merados
brew install merados-healthchecker
```

## Verfügbare Formulas

| Formula | Beschreibung |
|---------|--------------|
| `merados-healthchecker` | Trading Portfolio Healthcheck Tool |

## Verwendung

```bash
merados-healthcheck                  # Standard-Analyse
merados-healthcheck --user 13997     # Nur User 13997
merados-healthcheck --json           # JSON Output
merados-healthcheck --help           # Alle Optionen
```
